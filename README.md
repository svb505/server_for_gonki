## About: Server for race
## Features:
  - The server sends chat messages to all players.
  - The server moves players.
  - The server can roll back changes if the internet connection is poor.
  - The server stores the state history (**lag compensation**)
  - The server is authoritative (it makes all the decisions).
  - The server uses a snapshot system (sending data every frame and processing it).
  - Data is sent as data structures.
  - Protection against desynchronization (**server reconciliation**)
